🚀 Quick Start Guide

Get started with Hacker Simulator 2077 in 5 minutes.

📦 Installation

Requirements

  • Python 3.8 or higher
  • pip (Python package manager)
  • Git (optional, for cloning)

Step 1: Download the game

Option A: Clone with Git

git clone https://github.com/DlopedDtorred/hacker-simulator.git
cd hacker-simulator

Option B: Download ZIP

  1. Go to GitHub
  2. Click "Code" → "Download ZIP"
  3. Extract the ZIP file

Step 2: Install dependencies

pip install colorama
💡 Tip: It's recommended to use a virtual environment: 
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

Step 3: Run the game

python cyberdex.py

⚙️ First-Time Setup

When you run the game for the first time, you'll see:

  1. Language selection — Press 1 for Spanish, 2 for English
  2. Hacker name — Enter your name (or press Enter for "Zero_Cool")

Your configuration is saved in config.json.

🎮 Basic Controls

KeyAction
1Hack server (start a mission)
2Daily mission (DOUBLE rewards)
3Train (practice mini-games)
4Shop (buy tools)
5Statistics (view progress)
6Ranking (hacker rankings)
7Settings (theme, language, delete account)

9User guide
0Exit game

🎯 Your First Mission

  1. Press 1 to see available servers
  2. Choose a server (start with difficulty 1-3)
  3. Evade the firewall: Memorize and repeat a number sequence
  4. Crack the password: Use the clue to guess the password
  5. Earn credits and EXP!
💡 Tip: Read the clues carefully — they tell you exactly what the password is!

🛠️ First Tools to Buy

  • 🔧 Advanced Scanner (100 credits) — +2 password attempts
  • 🛡️ Firewall Bypass (200 credits) — Firewalls -1 difficulty
  • 🔑 Crypto Key (300 credits) — Reveals extra clues

💾 Saving Your Game

Press 8 at any time to save. Your progress is stored in save.json.

✅ Done! You're ready to play. Check the Beginner's Guide for more detailed tips.
