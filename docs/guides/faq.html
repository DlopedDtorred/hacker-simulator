❓ FAQ & Troubleshooting

Common questions and solutions for common problems.

❓ Frequently Asked Questions

¿Cómo guardo mi partida?

Presiona 8 en el menú principal. Tu progreso se guarda en save.json.

¿Cómo cambio el idioma?

Ve a Ajustes (7) → Cambiar idioma (2) y selecciona Español o English.

¿Cómo consigo créditos?

Completa misiones, misiones diarias y desbloquea logros. También puedes comprar EXP con créditos en la tienda.

¿Puedo jugar sin conexión?

Sí, el juego es 100% offline. Solo necesitas conexión para descargar actualizaciones.

¿Es gratuito?

Sí, es completamente gratuito y open source bajo licencia MIT.

¿Cómo contribuyo al proyecto?

Haz fork del repositorio, añade mejoras y abre un Pull Request. Revisa nuestra guía de contribución.

¿Qué hago si el juego no arranca?

Verifica que tienes Python 3.8+ instalado. Ejecuta pip install colorama y luego python cyberdex.py.

¿Cómo elimino mi cuenta?

Ve a Ajustes (7) → Eliminar cuenta (3) y confirma escribiendo ELIMINAR.

¿Por qué no puedo hackear ciertos servidores?

Puedes hackear servidores hasta tu nivel + 1. Sube de nivel para desbloquear más servidores.

¿Qué pasa si fallo una misión?

Pierdes tu racha y sumas un fallo, pero puedes intentarlo de nuevo.

🔧 Troubleshooting

El juego no se inicia

Verifica que tienes Python instalado: python --version. Si no, instala Python 3.8+.

Error: No module named 'colorama'

Instala colorama: pip install colorama. Si usas entorno virtual, actívalo primero.

El juego se cierra al iniciar

Ejecuta el juego con python cyberdex.py 2>&1 | tee error.log y revisa el archivo error.log.

Los colores no se ven bien

Asegúrate de ejecutar el juego en una terminal que soporte colores. En Windows, usa PowerShell o CMD.

El archivo de guardado está corrupto

Copia el archivo save.json a save.json.bak y elimina el original. El juego creará uno nuevo.

No puedo escribir contraseñas

Asegúrate de que la terminal está enfocada. A veces Ctrl+C o Ctrl+Z pueden bloquear la entrada.

⚠️ Si el problema persiste: Abre un issue en GitHub con la descripción detallada del problema y los pasos para reproducirlo.
