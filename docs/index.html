Instalador para Linux / Mac / Windows

Descarga el instalador, ejecútalo y elige la versión que quieras instalar. El instalador se encarga de todas las dependencias.

Descargar Instalador

Ejecuta: python installer.py

🛠️ Cómo instalar

Sigue estos pasos para tener el juego en tu ordenador.

1

📥 Descarga el instalador

Haz clic en "Descargar Instalador" y guarda el archivo.

2

▶️ Ejecuta el instalador

Abre una terminal y ejecuta: python installer.py

3

⚡ Elige la versión

Selecciona "UI Version" para la interfaz gráfica.

4

🎮 ¡A jugar!

En Linux/Mac ejecuta hacker-ui. En Windows busca el acceso directo en el escritorio.
