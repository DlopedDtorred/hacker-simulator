HACKER SIMULATOR 2077 - USER GUIDE
==================================

What is Hacker Simulator 2077?
------------------------------
A terminal-based hacking game where you play as an ethical hacker. Your mission is to infiltrate servers, evade firewalls, crack passwords, and become the best hacker in the digital underworld.

Key Features
------------
- 20 unique servers with different difficulties
- 4 mini-games (Firewall, Cryptography, SQL Injection, Logic Puzzles)
- 25+ achievements with rewards
- Streak system with bonuses
- 9 tools to purchase
- Daily missions with double rewards
- Save/Load system in JSON
- 4 visual themes (Matrix, Cyberpunk, Classic, Dark)
- Spanish/English multi-language support

Installation
------------
Requirements: Python 3.8+ and pip

Option 1: Install from GitHub
git clone https://github.com/DlopedDtorred/hacker-simulator.git
cd hacker-simulator
python -m venv venv
source venv/bin/activate (On Windows: venv\Scripts\activate)
pip install colorama
python cyberdex.py

Option 2: Download executable (Linux)
Download from GitHub Releases
chmod +x HackerSimulator2077-Linux
./HackerSimulator2077-Linux

First Time Setup
----------------
1. Select your language: Press 1 for Spanish, 2 for English
2. Enter your hacker name (or press Enter for "Zero_Cool")
3. Your configuration is saved in config.json

Game Controls
-------------
1 - Hack server
2 - Daily mission (DOUBLE rewards)
3 - Train (practice mini-games)
4 - Shop (buy tools)
5 - Statistics (view your progress)
6 - Ranking (see hacker rankings)
7 - Settings (change theme, language, delete account)
8 - Save game
9 - User guide
0 - Exit game

Gameplay Guide
--------------
Phase 1: Hack a Server
1. Choose a server from the list (difficulty 1-20 stars)
2. Evade the Firewall: Memorize and repeat a sequence of numbers
3. Crack the Password: Read the clue and type the password
4. Extra Security (if difficulty >= 5): Complete an additional mini-game

Phase 2: Level Up
- Gain EXP from completing missions
- Even levels: New tool
- Every 3 levels: Credits bonus
- Every 5 levels: Special reward

Phase 3: Daily Missions
- One special mission per day
- DOUBLE rewards (credits and EXP)
- Resets every 24 hours

Tips for Beginners
------------------
1. Read the clues carefully - They tell you exactly what to look for
2. Start with easy servers - Difficulty 1-3 to build your credits
3. Buy the Advanced Scanner first - More attempts = easier passwords
4. Keep your streak - Bonuses add up quickly
5. Do daily missions - Double rewards every day

Tips for Advanced Players
-------------------------
1. Save credits for expensive tools - Omega Key is worth it
2. Maintain a 10+ streak - Maximum bonuses
3. Hack all servers - Legend achievement gives 5000 credits
4. Use Crypto Key - Extra clues when stuck
5. Train mini-games - Practice makes perfect

FAQ
---
Q: How do I save my game?
A: Press 8 in the main menu. Your game is saved to save.json.

Q: How do I change the language?
A: Press 7 (Settings) -> 2 (Change language) -> Select your language.

Q: How do I change the theme?
A: Press 7 (Settings) -> 1 (Change theme) -> Select a theme.

Q: How do I delete my account?
A: Press 7 (Settings) -> 3 (Delete account) -> Type DELETE to confirm.

Q: Why can't I hack certain servers?
A: You can only hack servers up to your level + 1. Level up to unlock harder servers.

Q: How do I get more credits?
A: Complete missions, daily missions, and achievements.

Q: What happens when I fail a mission?
A: You lose your streak and get a failure count. But you can try again.

Q: How do I use tools?
A: Tools are passive - they automatically apply their effects during missions.

Troubleshooting
---------------
Problem: Game won't start
Solution: Check Python version (3.8+), install colorama (pip install colorama), try python3 cyberdex.py

Problem: Game crashes
Solution: Run python cyberdex.py and check error messages

Problem: Save file corrupted
Solution: Backup save.json, delete it, restart game

Problem: Color not showing
Solution: Reinstall colorama: pip uninstall colorama && pip install colorama
