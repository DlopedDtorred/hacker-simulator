3. MINI-JUEGOS DE SQL INJECTION
Cómo funciona: El juego te pide que inyectes código SQL para acceder a una tabla.

La tabla puede ser:

users

data

admin

system

logs

config

credentials

✅ Códigos SQL Correctos (cualquiera funciona):
Código	Explicación
SELECT * FROM users WHERE 1=1; --	Consulta completa con condición siempre verdadera
' OR '1'='1	Inyección clásica: siempre verdadero
'; DROP TABLE users; --	Inyección destructiva (elimina tabla)
' UNION SELECT null,null,null--	Inyección UNION
' OR 1=1 --	Inyección con comentario
📝 La más usada y segura:
text
' OR 1=1 --
🎯 Ejemplo de uso:
sql
