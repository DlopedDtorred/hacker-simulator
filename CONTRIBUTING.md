# Guía para Colaboradores

## 🚀 Primeros pasos

1. Lee el README.md
2. Prueba el juego localmente
3. Busca issues con la etiqueta `good first issue`

## 📝 Reglas

- Usa Python 3.8+
- Sigue PEP 8 para el código
- Prueba tu código antes de hacer PR

## 🎯 Áreas de contribución

- **Servidores**: Añadir nuevos servidores en `SERVIDORES`
- **Mini-juegos**: Crear nuevos juegos
- **Herramientas**: Añadir nuevas herramientas
- **Interfaz**: Mejorar la interfaz visual

## 🐛 Reportar bugs

Abre un issue con la etiqueta `bug`.
